## Installation

- To get started with this project, import it into Android Studio or build from the command line with Gradle:
 
 `gradlew assembleDebug` or `./gradlew assembleDebug`

---

## Features

- Modern, clean UI
- Blocks ads and trackers
- Extensions support including sideloading using urls and file
- Tab groups (inspired by Opera)
- Contextual bottom toolbar which shows buttons depending on context
